# Roast my code

Sample typescript code for the roast my code talk

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Npm](https://www.npmjs.com/)

### How to get started

- Clone the repo
- Run `npm install`
- Run `npm build`

## Project structure

- `src` - Source code
- `build` - Compiled code
